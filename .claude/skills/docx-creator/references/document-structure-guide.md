# Document Structure Guide

## Document Type Templates

### Report
1. **Title Page** - Document title, author, date, version
2. **Executive Summary** - 1 page max, key findings and recommendations
3. **Table of Contents** - Auto-generated from headings
4. **Introduction** - Purpose, scope, methodology
5. **Background** - Context and prior work
6. **Analysis** - Main content with data and findings
7. **Conclusions** - Summary of results
8. **Recommendations** - Actionable next steps
9. **Appendix** - Supporting data, glossary, references

### Meeting Minutes
1. **Meeting Information** - Title, date, time, location
2. **Attendees** - Present, absent, guests
3. **Agenda** - Numbered list of topics
4. **Discussion** - Key points per agenda item
5. **Decisions** - Formally agreed items
6. **Action Items** - Who, what, by when (table format)
7. **Next Meeting** - Date, time, location

### Proposal
1. **Cover Page** - Title, client name, date
2. **Executive Overview** - 1-paragraph summary
3. **Problem Statement** - Current challenges
4. **Proposed Solution** - Detailed approach
5. **Methodology** - How the work will be done
6. **Timeline** - Milestones and deliverables (table)
7. **Budget** - Cost breakdown (table)
8. **Team** - Key personnel and qualifications
9. **Summary** - Call to action

### Manual / Guide
1. **Cover Page** - Title, version, date
2. **Revision History** - Version log (table)
3. **Table of Contents**
4. **Introduction** - Purpose and audience
5. **Getting Started** - Prerequisites, setup
6. **Main Content** - Step-by-step procedures (numbered)
7. **Troubleshooting** - Common issues and solutions
8. **FAQ** - Frequently asked questions
9. **Appendix** - Reference tables, glossary

## Formatting Standards

### Fonts
| Element | Font | Size | Style |
|---------|------|------|-------|
| Title | Yu Gothic / MS Gothic | 24pt | Bold |
| Heading 1 | Yu Gothic / MS Gothic | 18pt | Bold |
| Heading 2 | Yu Gothic / MS Gothic | 14pt | Bold |
| Heading 3 | Yu Gothic / MS Gothic | 12pt | Bold |
| Body | Yu Mincho / MS Mincho | 10.5pt | Regular |
| Caption | Yu Gothic | 9pt | Italic |

### Page Layout (A4)
- Page size: 210mm x 297mm
- Top margin: 30mm
- Bottom margin: 25mm
- Left margin: 25mm
- Right margin: 25mm
- Header: 15mm from top
- Footer: 15mm from bottom

### Numbering
- Headings: 1. / 1.1 / 1.1.1 (max 3 levels in body)
- Figures: Figure 1, Figure 2, ...
- Tables: Table 1, Table 2, ...
- Pages: Arabic numerals, bottom center

### Table Rules
- Header row: bold, shaded background
- Alternating row shading for readability
- Left-align text, right-align numbers
- Include units in column headers
