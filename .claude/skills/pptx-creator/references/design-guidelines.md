# Slide Design Guidelines

## Core Principles
1. **One Message Per Slide** - Each slide should convey a single clear idea
2. **Visual Hierarchy** - Title > Subtitle > Body > Supporting details
3. **Whitespace** - At least 20% of slide area should be empty
4. **Consistency** - Same fonts, colors, and layout patterns throughout

## Typography
- **Title font**: 28-36pt, bold
- **Subtitle**: 20-24pt
- **Body text**: 16-20pt
- **Footnotes/captions**: 10-12pt
- Maximum 2 font families per presentation
- Prefer sans-serif fonts for screen presentations

## Color Palette
- **3-color rule**: main, accent, text
- Main color: used for titles, key elements
- Accent color: used for emphasis, charts, icons
- Text color: dark gray (#333333) or near-black, never pure black
- Background: white or very light neutral

## Layout
- **16:9 aspect ratio** (13.333 x 7.5 inches) is the standard
- Safe margins: at least 0.5 inches from all edges
- Align elements to an invisible grid
- Left-align text (avoid center-align for body text)

## Charts & Data
- Label axes and data points clearly
- Use color to distinguish series, not patterns
- Remove chart junk (unnecessary gridlines, borders, 3D effects)
- Maximum 5-6 data series per chart

## Images
- Use high-resolution images only (300 DPI for print, 150+ for screen)
- Maintain aspect ratio - never stretch
- Use consistent image treatment (all rounded corners, or all sharp, etc.)

## Slide Types Reference
| Type | Layout | Use Case |
|------|--------|----------|
| Title | Centered, large text | Opening/closing slides |
| Content | Title + bullets | Main information delivery |
| Two-column | Split layout | Comparisons, before/after |
| Chart | Title + chart | Data visualization |
| Table | Title + table | Structured data |
| Image | Full-bleed or title + image | Visual storytelling |
| Section divider | Large text, accent background | Topic transitions |
